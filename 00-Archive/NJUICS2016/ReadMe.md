实验框架代码： http://github.com/ics-nju-wl/icspa-public

实验指导书： http://github.com/ics-nju-wl/icspa-public-guide

**这个版本的 PA 有一个很有特色地方就是 PA1 是模拟加法器和浮点运算，可以很好的巩固课上学习的知识。即使选择了其他版本的 PA，也建议做一下 PA1。另外，测试加法器和浮点运算代码框架是一个很好的例子。**
