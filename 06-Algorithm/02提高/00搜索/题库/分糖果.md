### 题目描述
  
小明从糖果盒中随意抓一把糖果，每次小明会取出一半的糖果分给同学们。

当糖果不能平均分配时，小明可以选择从糖果盒中（假设盒中糖果足够）取出一个糖果或放回一个糖果。

小明最少需要多少次（取出、放回和平均分配均记一次），能将手中糖果分至只剩一颗。

### 输入描述
一个数 $n$，表示抓取的糖果数 $(n\leqslant 10^{10})$
## 输出描述
分至只剩一颗糖果最少需要多少次

### 输入用例
```
15
```

### 输出用例

```
5
```

### 说明

```
15+1=16;

16/2=8;

8/2=4;

4/2=2;

2/2=1;
```
