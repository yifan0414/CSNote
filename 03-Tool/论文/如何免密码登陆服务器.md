在自己的电脑上输入下面的命令，邮箱就输入自己邮箱就行，不重要
```bash
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

- 如果可以执行 `ssh-copy-id` 命令，然后执行下面的命令，例如把 ` user@server_ip ` 替换为 ` yifan@172.21.44.11 `
```bash
ssh-copy-id -i ~/.ssh/id_rsa.pub user@server_ip
```
