首先使用 `mkdir ～/clash` clash 命令，然后使用 `cd ~/clash` 命令。

- 在 `https://github.com/DustinWin/proxy-tools/releases/tag/Clash-Premium` 中下载 `clashpremium-release-linux-amd64.tar.gz`，并通过 `stfp` 工具将该压缩包上传到服务器的 `clash` 文件夹中，
	1. 使用 `tar -xvf clashpremium-release-linux-amd64.tar.gz` 解压得到 `CrashCore`，
	2. 然后通过 `mv CrashCore clash` 将解压出的可执行文件重命名。

- 在 `clash` 目录中执行 `wget -O config.yaml "https://7m8tk.no-mad-world.club/link/YHTw7AZifDT4Jeqc?clash=3"`
- 使用 `chmod +x clash`
- 使用 `./clash -d .` 启动 clash（如果需要长久保持，使用 nohup 或者 tmux 或者新建窗口）

如果需要使用 git，需要对 git 进行代理配置，命令如下
```bash
git config --global https.proxy http://127.0.0.1:7890
git config --global http.proxy http://127.0.0.1:7890
```
