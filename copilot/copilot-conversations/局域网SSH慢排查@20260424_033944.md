**user**: 请你告诉我 ssh 连接同一局域网的服务器慢怎么办
[Timestamp: 2026/04/24 03:39:44]

**ai**: 如果是**同一局域网下 SSH 很慢**，先优先怀疑 **Wi-Fi 网卡的省电模式** 导致的高延迟抖动。你的笔记里给出的处理办法是：

- 先在 Ubuntu 上执行 `iw dev`，查看无线网卡设备名[^1]
- 然后关闭该网卡的省电模式，例如：
  ```bash
  sudo iw wlp2s0 set power_save off
  ```
  其中 `wlp2s0` 要替换成你自己机器上查到的设备名[^1]

你也可以先用 `ping` 测一下局域网内服务器的延迟；如果 `ping` 延迟很高或波动很大，那么 SSH 慢很可能就是这个原因。你的笔记中这一节标题就是“同一个局域网下 ping 延迟过高”，并明确给出了上面这个关闭省电模式的方案。[^1]
